# PlanA

由于工作繁忙，此项目不再进行维护

## 功能描述

项目UI及功能主要模仿Google Keep笔记App，出发点是不喜欢Keep的所有任务都堆积在主页，不能按照日期(月份、星期)进行排版，因此萌生出自己开发的念头.

- [x] 按照日期对任务进行分类
- [x] 添加任务
- [x] 添加提醒任务
- [x] 添加已归档、垃圾桶
- [x] 添加撤销功能 
- [x] 添加常用模板
- [x] 对任务进行颜色区分
- [x] 提醒任务，闹铃或者通知栏通知
